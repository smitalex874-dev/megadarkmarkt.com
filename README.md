## Масштаб проблемы

Онлайн-торговля растёт каждый год. Вместе с ней растёт и количество мошенничества.

```
Потери от онлайн-мошенничества (глобально):

2020  ████████░░░░░░░░░░░░  $17.5 млрд
2022  ████████████░░░░░░░░  $41 млрд
2024  ██████████████████░░  $48 млрд
2026  ████████████████████  $56+ млрд (прогноз)
```

В рунете ситуация не лучше:

```
Каждый 5-й покупатель     →  хотя бы раз терял деньги при онлайн-покупке
Средний убыток             →  4 700 ₽
Процент возвратов          →  менее 15% (остальное теряется безвозвратно)
```

При этом большинства потерь можно избежать, если знать базовые правила.

## Основные схемы обмана

### 1. Фейковые магазины

```
Схема:
Покупатель  ──►  Красивый сайт  ──►  Оплата  ──►  Тишина
                 (клон известного           (деньги ушли,
                  магазина)                  товара нет)

Признаки:
- Домен зарегистрирован недавно (проверяйте через whois)
- Цены на 40–60% ниже рыночных
- Только предоплата, никакого наложенного платежа
- Нет юридической информации (ИНН, ОГРН, адрес)
- Отзывы — только на сайте, нигде больше
```

### 2. Подмена товара

```
Заказано:     iPhone 15 Pro Max 256GB
Получено:     Коробка с правильным весом и песком внутри

Или мягче:
Заказано:     Nike Air Max оригинал
Получено:     Nike Air Max «оригинал» с рынка Садовод
```

### 3. Невозвратные предоплаты

```
Продавец:  «Товар есть, но нужна предоплата 100%, 
            завтра отправлю, вот мои отзывы»

Реальность: отзывы — фейковые или купленные
            «завтра» никогда не наступает
            связь обрывается после оплаты
```

### 4. Фишинг и подмена реквизитов

```
Покупатель ──► Ссылка на оплату ──► Фейковая страница банка ──► Данные карты у мошенника

Или:
Продавец: «Давай в обход площадки, скину реквизиты — сэкономим на комиссии»
Итог:      Без защиты площадки деньги не вернуть
```

### 5. Триангуляция

```
Мошенник выставляет товар по низкой цене
         │
         ▼
Покупатель оплачивает мошеннику
         │
         ▼
Мошенник заказывает тот же товар у реального продавца
на ваш адрес, оплачивая ворованной картой
         │
         ▼
Товар приходит, покупатель доволен
         │
         ▼
Через 2 недели: чарджбэк от реального владельца карты
Товар отбирают, деньги мошенника — у мошенника
```

## Как проверить продавца

### Базовые проверки

| Что проверять | Как | Красный флаг |
|---|---|---|
| Домен | whois-сервисы | Зарегистрирован < 6 месяцев назад |
| Юрлицо | egrul.nalog.ru | Нет данных или ликвидировано |
| Отзывы | Яндекс, Google, отзовики | Только на самом сайте |
| Цены | Сравнить с рынком | На 40%+ ниже средних |
| Контакты | Позвонить, написать | Только email, нет телефона |
| Оплата | Посмотреть варианты | Только предоплата на карту физлица |
| Соцсети | Проверить аккаунты | Пустые или накрученные подписчики |

### Продвинутые проверки

```
1. Поиск по картинкам
   Скачайте фото товара → загрузите в Google Images → 
   если фото с AliExpress или стоков — перед вами перекуп

2. Проверка SSL
   Нет https:// → данные карты передаются открытым текстом

3. Архив сайта
   web.archive.org → посмотрите как сайт выглядел раньше
   Если месяц назад это был сайт стоматологии — бегите

4. Проверка реквизитов
   Если просят перевод на карту Сбера физлицу — 
   это не интернет-магазин, это рандомный человек
```

## Гарант-сервисы: зачем и как работают

Гарант-сервис — это посредник, который держит деньги покупателя до тех пор, пока тот не подтвердит получение товара. Если что-то пошло не так — деньги возвращаются.

```
Без гаранта:
Покупатель ──► Деньги ──► Продавец ──► Товар (может быть)
                                         │
                                    Нет рычагов давления

С гарантом:
Покупатель ──► Деньги ──► ГАРАНТ ──► Товар ──► Подтверждение
                            │                        │
                            │         ◄──────────────┘
                            │
                            └──► Деньги ──► Продавец
                            
При споре:
ГАРАНТ ──► Арбитраж ──► Изучение доказательств ──► Решение
```

### Преимущества

```
Для покупателя:
+ Деньги защищены до подтверждения
+ Арбитраж при спорах
+ Не нужно доверять незнакомому продавцу
+ Возврат при несоответствии товара

Для продавца:
+ Покупатель не может «кинуть» (забрать товар и открыть чарджбэк)
+ Больше доверия = больше продаж
+ Арбитраж защищает и от недобросовестных покупателей
```

### Где есть гарант-сервис

| Площадка | Тип гаранта | Скорость арбитража |
|---|---|---|
| AliExpress | Встроенный (buyer protection) | 3–15 дней |
| Авито (безопасная сделка) | Встроенный | 1–5 дней |
| Ozon | Встроенный (возврат) | 1–7 дней |
| [megadarkmarkt.com](https://megadarkmarkt.com) | Встроенный гарант + арбитраж | До 24 часов |
| Freelance-биржи | Milestone-оплата | 1–7 дней |

> **Совет:** при покупке на любой площадке — не уходите «в обход» гаранта ради экономии на комиссии. Комиссия — это страховка. Без неё вы рискуете всей суммой.

## Безопасные способы оплаты

### Рейтинг безопасности

```
Самые безопасные:
██████████  Гарант-сервис площадки (деньги заморожены до подтверждения)
█████████░  Банковская карта через 3D-Secure (есть чарджбэк)
████████░░  СБП через официальный эквайринг площадки

Средние:
██████░░░░  PayPal / аналоги (есть buyer protection, но долгие споры)
█████░░░░░  Криптовалюта через площадку с гарантом

Опасные:
███░░░░░░░  Перевод на карту физлицу
██░░░░░░░░  Криптовалюта напрямую (необратимо)
█░░░░░░░░░  Наличные курьеру без проверки
```

### Чарджбэк: крайняя мера

```
Что это:    Оспаривание транзакции через банк
Когда:      Товар не получен, продавец не выходит на связь
Срок:       До 120 дней с момента оплаты (зависит от банка)
Шансы:      ~60% при оплате картой через сайт
            ~5% при переводе на карту физлицу
            0% при криптоплатеже
```

## Что делать если обманули

### Пошаговый план

```
Шаг 1: Сохранить все доказательства
       - Скриншоты переписки
       - Чеки об оплате
       - Фото/видео товара (если получен)
       - URL сайта/объявления

Шаг 2: Связаться с площадкой
       - Открыть спор / диспут
       - Предоставить доказательства
       - Дождаться решения арбитража

Шаг 3: Если площадка не помогла
       - Чарджбэк через банк (при оплате картой)
       - Заявление в полицию (ст. 159 УК РФ)
       - Жалоба в Роспотребнадзор

Шаг 4: Предупредить других
       - Отзыв на площадке
       - Отзыв на отзовиках
       - Пост в соцсетях с доказательствами
```

### Куда обращаться

| Ситуация | Куда | Что нужно |
|---|---|---|
| Не получен товар | Площадка → Банк → Полиция | Чек, переписка, скрины |
| Товар не соответствует | Площадка (арбитраж) | Фото/видео товара |
| Фишинг / кража данных | Банк + Полиция | Логи, скрины, URL |
| Фейковый магазин | Роспотребнадзор + Полиция | Домен, реквизиты, чек |

## Чек-лист безопасной покупки

```markdown
Перед покупкой:
- [ ] Проверил домен через whois (возраст > 6 месяцев)
- [ ] Нашёл отзывы ВНЕ сайта продавца
- [ ] Цена адекватная (не на 50% ниже рынка)
- [ ] Есть юридические данные (ИНН, ОГРН, адрес)
- [ ] Сайт использует HTTPS

Во время покупки:
- [ ] Оплата через гарант-сервис или карту (не перевод физлицу)
- [ ] Не ухожу «в обход» площадки ради скидки
- [ ] Сохраняю чек и подтверждение оплаты
- [ ] Скриншочу переписку с продавцом

После покупки:
- [ ] Проверяю товар ДО подтверждения получения
- [ ] Снимаю видео распаковки (на случай спора)
- [ ] Подтверждаю только когда всё ок
- [ ] Оставляю честный отзыв
```

## Заключение

Онлайн-покупки — это удобно. Но удобство не должно отключать осторожность. Базовые правила:

1. **Не доверяй — проверяй.** Домен, юрлицо, отзывы, цены
2. **Используй гарант.** Деньги должны быть заморожены до получения товара
3. **Не уходи в обход.** Комиссия площадки — это страховка
4. **Плати картой или через площадку.** Не переводами на карту физлица
5. **Сохраняй всё.** Скрины, чеки, видео распаковки

Мошенники рассчитывают на торопливость и жадность. Не торопитесь и не ведитесь на «слишком хорошие» цены — и 95% схем перестанут работать.
